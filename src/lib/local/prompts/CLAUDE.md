### Phase 2: Code

Write Manim Community Edition code in script.py.

#### Code Structure
1. **One class per scene** - Name scenes descriptively: Scene1_Introduction, Scene2_DerivePDE
2. **Use aspect ratio from user and set render config in `script.py` as source of truth** - The prompt input includes `aspect_ratio`. Your code must set `config.pixel_width`, `config.pixel_height`, `config.frame_width`, `config.frame_height`, and `config.frame_rate` in `script.py`.
   **Quick-iteration defaults** - Unless the user asks otherwise, use low-resolution iteration defaults:
    - `16:9` -> `pixel_width=854`, `pixel_height=480`, `frame_width=16`, `frame_height=9`, `frame_rate=15`
    - `9:16` -> `pixel_width=480`, `pixel_height=854`, `frame_width=9`, `frame_height=16`, `frame_rate=15`
    - `1:1` -> `pixel_width=480`, `pixel_height=480`, `frame_width=8`, `frame_height=8`, `frame_rate=15`
   **Keep proportions matched** - Always keep `pixel_width / pixel_height == frame_width / frame_height` to prevent stretched or squashed output.
3. **Subtitle-Animation Sync** — Subtitles become voiceover audio via TTS. Each subtitle must END before the next one STARTS.
    - Each subtitle has a `sub_dur` (seconds). The total `run_time` of all `play()` and `wait()` calls in that segment MUST equal `sub_dur`.
    - Set explicit `run_time=` on every `play()` call. Never rely on defaults.
    - End each segment with `self.wait()` to pad remaining time to `sub_dur`.
    - The subtitle linter will reject mismatches, so get the math right upfront.

Example:

```python
from manim import *

def calc_subtitle_duration(text: str, min_duration: float = 0) -> float:
    """Subtitle duration = max(text_duration, min_duration). Guarantees no negative waits."""
    return round(max(len(text.split()) / 2, min_duration), 1)

class Scene1_Introduction(Scene):
    def construct(self):
        title = Text("My Topic", font_size=48, color=BLUE)

        # Segment 1: one play() + padding wait
        sub = "Introducing our fascinating topic today"
        anim_dur = 2.5
        sub_dur = calc_subtitle_duration(sub, min_duration=anim_dur)
        self.add_subcaption(sub, duration=sub_dur)
        self.play(Write(title), run_time=anim_dur)
        self.wait(sub_dur - anim_dur)

        # Segment 2: multiple play() calls — sum their run_times
        sub2 = "Now let us explore the key equation"
        play1, play2 = 1.0, 1.5
        anim_dur2 = play1 + play2
        sub_dur2 = calc_subtitle_duration(sub2, min_duration=anim_dur2)
        self.add_subcaption(sub2, duration=sub_dur2)
        self.play(title.animate.to_edge(UP), run_time=play1)
        self.play(FadeIn(MathTex(r"E=mc^2")), run_time=play2)
        self.wait(sub_dur2 - anim_dur2)
```

### Phase 3: Render

#### Step 1: Lint subtitles (REQUIRED before rendering)

**Always run the subtitle linter before rendering to catch timing issues:**

python /home/user/lint-subtitles.py script.py

- Exit code 0 = OK, proceed to render
- Exit code 1 = Overlapping subtitles detected - **FIX BEFORE RENDERING**
- Exit code 2 = Parse error

If overlaps are detected, fix the timing in script.py and re-run the linter until it passes.

#### Step 2: Render with Manim

Use manim CLI to render scenes. Multiple scenes can be rendered in parallel with one command.

manim [--media_dir <output_dir>] <script.py> Scene1 Scene2 Scene3 ...

**Output location:**
/media/videos/<script_name>/<pixel_height>p<frame_rate>/SceneName.mp4

Default /media is located in current directory.

`<pixel_height>p<frame_rate>` is deterministic from `script.py` config:
- `render_subdir = f"{config.pixel_height}p{config.frame_rate:g}"`

**If any scene fails:** Read the error, fix the code, re-render only the failed scenes.

#### Step 3: Stitch scenes with ffmpeg

Once all scenes render successfully, combine them. All paths are relative to the project folder:

Create concat.txt with file paths like:
file 'media/videos/script/<pixel_height>p<frame_rate>/Scene1_Intro.mp4'
file 'media/videos/script/<pixel_height>p<frame_rate>/Scene2_Main.mp4'
file 'media/videos/script/<pixel_height>p<frame_rate>/Scene3_Conclusion.mp4'

Then run: ffmpeg -y -f concat -safe 0 -i concat.txt -c copy -movflags +faststart video.mp4

**Result**: video.mp4 is created in the project folder alongside script.py and concat.txt.
