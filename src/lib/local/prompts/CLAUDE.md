### Phase 3: Code

Write Manim Community Edition code in script.py.

#### Code Structure
1. **One class per scene** â€” Name scenes descriptively: Scene1_Introduction, Scene2_DerivePDE
2. **Use the `**Aspect Ratio**` from the prompt unless the user prompt overrides it, and set render config in `script.py` as source of truth** â€” Your code must set `config.pixel_width`, `config.pixel_height`, `config.frame_width`, `config.frame_height`, and `config.frame_rate` in `script.py`.
   **Quick-iteration defaults** â€” Unless the user asks otherwise, use low-resolution iteration defaults:
    - `16:9` -> `pixel_width=854`, `pixel_height=480`, `frame_width=16`, `frame_height=9`, `frame_rate=15`
    - `9:16` -> `pixel_width=480`, `pixel_height=854`, `frame_width=9`, `frame_height=16`, `frame_rate=15`
    - `1:1` -> `pixel_width=480`, `pixel_height=480`, `frame_width=8`, `frame_height=8`, `frame_rate=15`
   **Keep proportions matched** â€” Always keep `pixel_width / pixel_height == frame_width / frame_height`.
3. **Use `duration_s` from timestamps.json as the total segment budget** â€” According to `timestamps.json`, inline the values as literals in `script.py`.
   - Set explicit `run_time=` on every `play()` call.
   - Keep total scheduled time within `duration_s`, including `wait()`, `move_camera()`, and any later `play()` such as `FadeOut`.
   - End each segment with `self.wait(max(0, duration_s - used_time))`. Always clamp to avoid negative waits.
4. **Use TeX for text rendering** â€” You MUST use `Tex(...)` for normal text and `MathTex(...)` for formulas. Avoid `Text(...)` unless there is a specific non-TeX reason, because Manim's text renderer can produce strange letter spacing.

Example:

```python
from manim import *

class Scene1_Introduction(Scene):
    def construct(self):
        title = Tex("My Topic", font_size=48, color=BLUE)

        dur = 2.8                              
        text = "Introduction to My Topic"    
        anim_rt = 1.5
        self.add_subcaption(text, duration=dur)
        self.play(Write(title), run_time=anim_rt)
        self.wait(max(0, dur - anim_rt))
```

### Phase 4: Render

#### Run subtitle lint first to ensure subtitles and animations match

```
python lint-subtitles.py script.py
```
Proceed to render only if it passes. Otherwise, modify the code.

#### Render with Manim

```
manim [--media_dir <output_dir>] <script.py> Scene1 Scene2 Scene3 ...
```

**Output location:** `media/videos/script/<pixel_height>p<frame_rate>/SceneName.mp4`

`<pixel_height>p<frame_rate>` is deterministic from `script.py` config:
- `render_subdir = f"{config.pixel_height}p{config.frame_rate:g}"`

**If any scene fails:** Read the error, fix the code, re-render only the failed scenes.

#### Stitch scenes with ffmpeg

Create concat.txt:
```
file 'media/videos/script/<pixel_height>p<frame_rate>/Scene1_Intro.mp4'
file 'media/videos/script/<pixel_height>p<frame_rate>/Scene2_Main.mp4'
```

Then run:
```
ffmpeg -y -f concat -safe 0 -i concat.txt -c copy -movflags +faststart video_silent.mp4
```

### Phase 5: Mux

Combine silent video with voiceover:

```
ffmpeg -y -i video_silent.mp4 -i voiceover.mp3 \
  -map 0:v:0 -map 1:a:0 \
  -c:v copy -c:a aac \
  -movflags +faststart video.mp4
```

**Result**: `video.mp4` is the final deliverable.
